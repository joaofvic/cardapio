# **App Name**: Harvest Bites

## Core Features:

- Curated Meal Browsing: Users can browse a collection of chef-curated meals, filtered by categories like 'Chicken', 'Beef', and 'Veggie', matching the design.
- Seasonal Spotlight Display: Showcase a featured meal bundle or plan, highlighting new seasons or high-protein options with detailed descriptions, similar to 'The Harvest Performance Bundle'.
- Detailed Meal Information: Present comprehensive details for each meal, including pricing, a full description, and key nutritional facts (protein, carbs, calories).
- Dynamic Search Tool: Enable users to search for meals or ingredients to easily find desired items.
- Add to Cart Functionality: Allow users to add selected meals to a shopping cart for order placement.
- Shopping Cart Management: Display items in the user's cart, showing quantities and enabling updates or removal of items before checkout.
- AI Meal Recommendation Tool: Leverage generative AI to suggest personalized 'Popular Favorites' meals to users based on their browsing activity and highlighted meal characteristics.

## Style Guidelines:

- Light color scheme with a prominent fresh green. The background is a very light desaturated green: #ECF6EE. The primary action color for buttons and highlights is a rich green: #0F8A25. A vibrant yellow-green accent is used for key active elements: #95E31D.
- Headlines and body text will use 'Inter', a clean and modern grotesque-style sans-serif, providing a professional and objective feel. This ensures readability for both short and longer text chunks.
- Utilize simple, line-based icons for navigation (Home, Menu, Cart) and UI elements (notification bell, search magnifying glass). Use solid fill icons for ratings (e.g., stars) and a prominent plus sign for adding items to the cart, consistent with the image's minimalistic aesthetic.
- A clean, card-based layout featuring a single-column scrollable feed for meal listings and a fixed bottom navigation bar for core functionalities. Ample use of whitespace will ensure clarity and focus on content. Featured elements like the 'Weekly Spotlight' will use large, immersive visuals.
- Implement subtle animations for interactive elements, such as hover effects on meal cards, smooth transitions when adding items to the cart, and slight shifts in the bottom navigation icons upon selection to provide visual feedback without distraction.
